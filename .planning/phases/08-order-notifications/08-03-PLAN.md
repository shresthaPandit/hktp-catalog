  <what-to-do>
Create the n8n workflow in n8n Cloud. Step-by-step:

**Step 1: Create workflow**
1. Open n8n Cloud → New Workflow → name it "Order WhatsApp Notification"

**Step 2: Add Webhook node (trigger)**
1. Add node: Webhook
2. HTTP Method: POST
3. Path: order-notification (or any unique path — the full URL is what matters)
4. Authentication: None (rely on env var URL secrecy)
5. Respond: Immediately (returns 200 without waiting for downstream nodes — this is what triggers the "ok" result in notifications.ts)
6. Switch to Production mode (not Test) — copy the Production URL

**Step 3: Add Code node to build the WhatsApp message**
Add a Code node after the Webhook node. JavaScript mode. Use this template:

```javascript
const body = $input.first().json.body;
const itemLines = body.items.map(item => {
  const price = item.price_at_order != null ? `$${item.price_at_order.toFixed(2)}` : 'Price TBD';
  const subtotal = item.subtotal != null ? ` = $${item.subtotal.toFixed(2)}` : '';
  return `• ${item.name} x${item.quantity} @ ${price}${subtotal}`;
}).join('\n');

const notesLine = body.customer_notes ? `\n📝 Notes: ${body.customer_notes}` : '';
const subtotalLine = body.subtotal > 0 ? `\n💰 Subtotal: $${body.subtotal.toFixed(2)}` : '';

const message = `🔔 *New Order: ${body.order_number}*\n\n` +
  `🏢 ${body.company_name}\n` +
  `📞 ${body.phone}\n\n` +
  `*Items:*\n${itemLines}` +
  subtotalLine +
  notesLine;

return [{ json: { message } }];
```

1. Add node: Twilio
2. Use existing Twilio credentials (from Phase 5 — already configured in n8n)
3. From: whatsapp:+14155238886 (TWILIO_WHATSAPP_FROM)
4. To: whatsapp:{admin_phone} — confirm the admin WhatsApp number and enter it here
5. Message: `{{ $json.message }}` (from Code node output)

**Step 5: Activate and copy URL**
1. Activate the workflow (toggle top-right)
2. Open the Webhook node → copy the Production URL
3. It will look like: https://your-instance.n8n.cloud/webhook/order-notification
