**NODE 18 — Code: Check Failure Alert**
- Type: Code
- Mode: Run Once for All Items
- JavaScript:
```javascript
const rows = $input.all();
const staticData = $getWorkflowStaticData('global');

// Check if the last 3 runs all failed
const allFailed = rows.length === 3 && rows.every(r => r.json.status === 'failed');

// Deduplication: only alert once every 2 hours to avoid spam
const lastAlertAt = staticData.lastAlertSentAt
  ? new Date(staticData.lastAlertSentAt)
  : new Date(0);
const twoHoursAgo = new Date(Date.now() - 2 * 60 * 60 * 1000);
const cooldownPassed = lastAlertAt < twoHoursAgo;

const shouldAlert = allFailed && cooldownPassed;

if (shouldAlert) {
  staticData.lastAlertSentAt = new Date().toISOString();
}

return [{ json: { shouldAlert } }];
```
→ Connects to NODE 19
